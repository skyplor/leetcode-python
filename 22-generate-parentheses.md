# 22. Generate Parentheses
_medium_

Given `n` pairs of parentheses, write a function to _generate all combinations of well-formed parentheses_.

### Example 1:

<pre>
<b>Input:</b> n = 3
<b>Output:</b> ["((()))","(()())","(())()","()(())","()()()"]
</pre>

### Example 2:

<pre>
<b>Input:</b> n = 1
<b>Output:</b> ["()"]
</pre>

### Constraints:

- `1 <= n <= 8`
